# Instructions
## Setup
### Dependencies
Create a virtual environment callled venv
install the following packages using pip:
 - mako
 - cherrypy

### DB
Create new ./db/seed.db using:

- ./db/create-db.sql
- ./db/populate-db.sql

## Run
python3 seedb.py

Navigate to
http://127.0.0.1:8080
